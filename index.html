Web Design
Supermarket Application

Supermarket Management Application

Project Description

Supermarket management application aims at developing a computerized system to aid in the handling and simplification of the daily activities of a supermarket. The user is faced with an initial login page where he/she has to enter his/her credentials. Depending on the designation of the user he/she is granted access to selective operations of the supermarket. A multitude of facilities are provided by this system. There is a provision to view, search for, add, delete, modify and manage inventory and stock available in the store, employees of the supermarket, and the customers of the establishment. Additionally, those who have the appropriate authority can also perform billing, as well as delete and view the invoices of previous purchases. This is implemented through various interfaces and options provided after successful login. Selective functionality is provided after login to all users depending on their designation; for instance only a cashier and manager can perform billing, while a generic employee can only access inventory. Overall this project is being developed to greatly reduce all the efforts that are involved in the smooth functioning of a supermarket.

Duration: August-November,2018.

Technology: HTML, CSS, JavaScript, PHP.

Database: MySQL.
